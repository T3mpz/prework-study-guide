Hyper Text Markup Langauge (HTML)

  • HTML provides structure for a webpage.
  • Elements are used to divide content on the page.
  • Semantic HTML elements describe content and the role it plays on a webpage.
  • Head element contains information (metadata) which helps the browers render correctly.
  • Body element contains the content of the webpage shown to the user.
  • Attributes define additonal informaoin about an element.

Cascading Styling Sheet (CSS)

  • CSS is a programming language used by developers to define how webpage content is presented to users.
  • A margin property indicates how much space we want around the outside of an element
  • The padding property adds space around the content inside an element
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.

Git & GitHub

  • Git is a version control system that allows you to merge and track different versions of code.
  • GitHub is a cloud-based hosting service that allows us to track web projects inside remote repos.
  • Clone (copy), edit (revise code), stage (git add -A), commit (git commit -m), push (git push)
